# DnD Ruby Web App

A learning project building a D&D campaign tracker web application with Ruby.

## Overview

This project demonstrates full-stack Ruby web development by creating an interactive platform to document and manage D&D playthroughs.

## Tech Stack

- **Language**: Ruby
- **Database**: PostgreSQL
- **Styling**: Tailwind CSS
- **CI/CD**: GitHub Actions

## Features

- Create and manage D&D campaigns
- Track player characters and progression
- Log session notes and encounters
- Collaborative playthrough documentation

## Getting Started

### Prerequisites

- Ruby 3.0+
- PostgreSQL
- Node.js (for Tailwind)

### Installation

```bash
git clone <repository>
cd DnD-Ruby
bundle install
rails db:create
rails db:migrate
```

### Running the App

```bash
./bin/dev
```

Visit `http://localhost:3000`

## Development

This project uses GitHub Actions for automated testing and deployments.
